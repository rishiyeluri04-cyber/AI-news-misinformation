PRESENTATION

TruthLens

AI-Based Fake News Detection System

Combating Misinformation with Machine Learning & NLP

1. Introduction

  • AI/NLP driven detection for high-stakes news environments.
  • Automated classification as Real or Fake.
  • Objective: Neutralize misinformation at the source.

2. Problem Statement

  • Digital platforms have exponentialized the spread of 'Fake News'.
  • Manual fact-checking is too slow for 21st-century information cycles.
  • Need for high-volume, automated verification tools.

3. Objectives

  • Real-time prediction and categorization.
  • High precision and recall for investigative use.
  • Promote a responsible, truth-driven digital ecosystem.

4. System Architecture

  • Input Processing → Text Cleaning (NLTK)
  • Vectorization (TF-IDF) → Dimensionality Reduction
  • Inference (ML Ensemble) → Gemini AI Verification
  • Dynamic UI Output View.

5. Functional Requirements

  • Multi-format support (Direct Text & File Uploads).
  • Comprehensive Preprocessing (Stemming/Stop-words).
  • Dual-Verification (Traditional ML + Generative AI).

6. Non-Functional Requirements

  • Speed: Latency < 3 seconds globally.
  • Accuracy: Minimum 85% validation threshold.
  • Design: Modern Responsive UI with sleek aesthetics.

7. Technology Stack

  • Frontend: Next.js 14, React, Framer Motion.
  • Backend: Python 3.10, Flask Micro-framework.
  • Core: Scikit-learn, Pandas, NumPy, Gemini 1.5 API.

8. Performance Metrics

  • Primary Metric: Overall Accuracy.
  • Secondary: Precision, Recall, and F1-Score.
  • Explainability: Keyword contribution mapping.

9. Future Enhancements

  • Transformer integration (BERT/GPT Fine-tuning).
  • Global Language Support (i18n).
  • Browser Extension & Social Media bot integration.

10. Conclusion

  • TruthLens provides a scalable, reliable shield against misinformation.
  • Automated truth-detection is no longer a luxury, but a necessity.
  • The future of journalism is AI-augmented and Fact-verified.
